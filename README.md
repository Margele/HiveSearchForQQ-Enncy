# HiveSearchForQQ
### 网课搜题QQ机器人

## 使用方法
**输入：**<br>
**q问题**<br>
**即可收到答案回复**

**例如：**<br>
**q我爱你**

## 所有关键字

| 触发词 | 含义   |
|-----|------|
|  搜题帮助  | 帮助提示 |
|  开启搜题  | 当前群搜索功能开启 |
|  关闭搜题  | 当前群搜索关闭关闭 |
|  开启所有人可变更搜题权限   |  所有人权限开启 |
|  关闭所有人可变更搜题权限   |  所有人权限关闭 |
